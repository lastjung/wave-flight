# 게임 방법

## 기본 조작
- 이동: W/↑ 위, S/↓ 아래, A/← 왼쪽, D/→ 오른쪽
- 부스트: Shift
- 발사: Space
- 탄환 전환: Q
- 모바일/터치: 화면을 드래그해서 조종

## 모바일 사용
- HUD 버튼(오른쪽 위)로 설정 패널 열기/닫기
- FIRE 버튼으로 발사

## 탄환 타입
- NORMAL: 기본 탄환
- PIERCE: 최대 5회 관통, 데미지 낮음
- BLAST: 범위 폭발 피해
- EMP: 적을 잠시 느리게(스턴)
- SHOTGUN: 10도 스크류 확산 다발 발사
- LASER: 직선 빔으로 다중 히트

## 목표
- 장애물/적을 피하거나 파괴하면서 점수를 올린다.
- 연료가 바닥나면 게임 오버.
