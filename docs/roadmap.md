# Future Roadmap: Steampunk Wave Flight

프로젝트의 다음 단계를 위한 아이디어 제안입니다.

## 1. 비주얼 및 연출 강화 (Visual Polish)

- [ ] **파티클 시스템 고도화**: 장애물 충돌 시 단순 점멸 대신 파편이 튀거나 폭발하는 연출 추가.
- [ ] **다양한 장애물**: 현재의 단순한 큐브/구 형태에서 톱니바퀴, 기뢰, 떠다니는 섬 등 스팀펑크 테마에 맞는 모델로 교체.
- [ ] **환경 변화**: 비행 거리에 따라 하늘 색상이 변하거나(낮/밤/노을), 안개 농도가 짙어지는 등의 환경 변화 구현.

## 2. 게임플레이 심화 (Gameplay Depth)

- [ ] **연료/스팀 게이지**: 부스트(Shift) 사용 시 소모되는 자원(Steam Pressure)을 추가하고, 이를 관리하는 전략 요소 도입.
- [ ] **아이템(Collectibles)**: 연료 보급, 점수 보너스, 일시적 무적 아이템 등을 공중에 배치.
- [ ] **난이도 곡선**: 일정 점수/거리 도달 시 속도가 빨라지거나 장애물 밀도가 높아지는 레벨 디자인.

## 3. 기술적 개선 (Technical Improvements)

- [ ] **모바일 컨트롤 개선**: 현재의 드래그 방식 외에 가상 조이스틱 구현으로 조작감 향상.
- [ ] **코드 리팩토링**: `main.js`의 비대화를 막기 위해 `Game` 클래스를 더 작은 매니저(UIManager, GameManager)로 분리.
- [ ] **설정 저장**: 로컬 스토리지를 활용하여 볼륨, 최고 점수(High Score) 등을 브라우저에 저장.

## 4. 사운드 확장 (Audio Expansion)

- [ ] **상황별 BGM**: 위기 상황(체력 부족)이나 고속 비행 시 음악의 템포나 레이어가 바뀌는 적응형 음악(Adaptive Music).
- [ ] **UI 사운드**: 버튼 클릭, 슬라이더 조절 시의 기계적인 효과음(Click, Clank) 추가.
