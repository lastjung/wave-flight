# 요구사항(문자 변환)

> "Three.js와 Perlin Noise 알고리즘을 활용해 80년대 레트로웨이브 스타일의 보라색 네온 그리드 지형이 카메라를 향해 끝없이 다가오는 비행 시뮬레이션을 구현하되, 지형의 높낮이가 실시간으로 춤추듯 역동적으로 변하고 배경에는 거대한 레트로 태양이 서서히 지며 붉은빛을 내뿜는 몽환적인 장면을 만들어줘. 서브폴더에 개발. 프람프트 개발내용 저장"

## 조건
- Three.js 사용
- Perlin Noise로 지형 높낮이 실시간 변화
- 80년대 레트로웨이브: 보라색 네온 그리드 지형 + 거대한 레트로 태양(서서히 지는 연출)
- 단일 HTML(HTML/CSS/JS 포함)
- 같은 서브폴더에 프롬프트/개발내역 저장
